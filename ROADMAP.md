# FieldSense Roadmap

This roadmap keeps the project focused on practical field-team workflows.

## Near Term

- Add clearer deployment docs for Cloudflare Workers and Supabase.
- Add sample `.env.example` explanations for every required variable.
- Add seed data scripts for demo field visits.
- Add screenshots for login, dashboard, new visit, visit history, media upload, and Ask AI.
- Add basic API route tests for health, visits, debrief, and RAG query.

## Product Improvements

- Offline-first draft capture for low-connectivity field visits.
- Better media preview and upload progress states.
- Export visit records as CSV or PDF.
- Add program-specific templates for skilling, livelihood, health, education, and community outreach visits.
- Add follow-up task tracking with status and owner fields.

## AI and RAG Improvements

- Add retrieval evaluation examples with expected supporting visits.
- Add configurable chunking for long visit notes.
- Add citation-style source display in Ask AI answers.
- Add safeguards for low-confidence answers.
- Add admin controls for choosing embedding model and reindex scope.

## Community

- Label beginner-friendly issues.
- Add demo data that does not contain private or sensitive information.
- Document privacy and responsible AI considerations for field data.
- Collect workflow feedback from NGO, public-interest, and student-builder users.
