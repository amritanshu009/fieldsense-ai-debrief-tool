# Changelog

All notable changes to FieldSense will be documented in this file.

## 0.1.0 - 2026-06-26

### Added

- Initial open-source release.
- Field visit creation with location, date, program area, stakeholders, notes, findings, blockers, follow-ups, and tags.
- Supabase PostgreSQL persistence.
- Supabase Auth login with role-based access.
- Supabase Storage media support for photos, videos, and voice memos.
- Gemini-based AI debrief generation.
- Gemini embeddings with Supabase `pgvector` semantic search.
- RAG-based Ask AI flow over saved visits.
- Admin-only reindexing.
- Dashboard insights for blockers, sentiment, and follow-up patterns.
